# Marginal Interface

Marginal Web App

## Installation

Install with yarn

```sh
yarn
```

## Requirements

Require a valid alchemy rpc stored in your .env file:

```sh
REACT_APP_SEPOLIA_ALCHEMY_RPC = YOUR_API_KEY
REACT_APP_CODEX = YOUR_API_KEY
```

## Local Development

To start local development on localhost:3000

```sh
yarn start
```
